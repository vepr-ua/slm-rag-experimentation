# Graph Module

SurrealDB interaction and graph traversal algorithms.

## Components

- `client.py` - SurrealDB connection and client
- `schema.py` - Database schema definitions
- `traversal.py` - Graph traversal algorithms (BFS/DFS, weighted paths)
- `queries.py` - Common graph queries

## Responsibilities

- Manage connections to SurrealDB
- Execute graph traversal for concept exploration
- Subgraph extraction around query entities
- Relationship strength calculation
